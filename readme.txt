Peonpegu(Pemesanan Online Pendakian Gunung) menggunakan PHP Native.

Template :

    Biznance by Colorlib
    Stisla Master (Admin)

Fitur :

    Reset kata sandi
    Daftar 

Fiture sebagai Admin :

    CRUD data berita.
    CRUD data wisata.
    CRUD data transaksi.
    CRUD data user.
    Laporan transaksi.
    Print transaksi via Excel & Pdf
    Login & Logout
    Update profile

Fiture sebagai User :

    Pesan tiket wisata
    Testimoni
    Aktivasi kode SMS OTP(dalam pengembangan)
    Update profile
    Print bukti pembayaran via Pdf
    Komentar 


Barangkali ada yang minat bisa inbox. Gratis., namun jika ada yang mau menyumbangkan sedikit rezeki nya pun juga tidak masalah akan saya terima hehe.
